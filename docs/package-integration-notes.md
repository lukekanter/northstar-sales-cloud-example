# Package integration notes

The Northstar build vendors five open-source accelerators from GitHub source
rather than installing them from AppExchange. This file is the permanent record
of **what** we cloned, **which commit** was reviewed, and **under what
license**. The clones themselves live in the gitignored `vendor/` directory;
re-create them with `./scripts/clone-vendor.sh`.

## Notes

- **Pinned SHAs** capture the exact upstream state reviewed on 2026-05-20.
  If a SHA is bumped intentionally, update the row here and the matching entry
  in `scripts/clone-vendor.sh` in the same change.
- **Licenses** are copied verbatim into `LICENSES/<accelerator>-LICENSE.txt`.
  Those files are the permanent attribution and must be retained even after the
  `vendor/` clones are deleted.
- **Source folder used** is the directory within each upstream repo that holds
  the metadata we care about.
